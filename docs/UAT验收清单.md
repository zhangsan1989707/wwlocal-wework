# UAT 验收清单

本清单用于首次上线、大版本升级或客户交付前验收。每项验收建议记录执行人、执行时间、测试账号、测试数据范围和实际结果。

| 编号 | 验收项 | 前置条件 | 操作步骤 | 通过标准 | 结果 |
|------|--------|----------|----------|----------|------|
| UAT-01 | 超级管理员登录 | 已配置可用管理员账号 | 打开前端地址，输入超级管理员账号密码登录 | 登录成功后进入运营总览；左侧可看到同步任务、用户管理、密钥管理等管理菜单 | □ 通过 / □ 不通过 |
| UAT-02 | 登录失败与限流 | 已准备错误密码 | 连续输入错误密码登录；达到限制后继续尝试 | 错误密码返回失败提示；短时间多次失败后触发临时锁定；等待解锁后可再次尝试 | □ 通过 / □ 不通过 |
| UAT-03 | 部门管理员权限 | 已创建部门管理员并配置授权部门 | 使用部门管理员登录，访问运营总览、查询、导出和用户明细 | 只能看到授权部门及子部门数据；不能访问用户管理、密钥管理等超级管理员菜单 | □ 通过 / □ 不通过 |
| UAT-04 | 越权查询拦截 | 已准备非授权部门成员手机号、UserID 或 OpenID | 部门管理员查询非授权成员日志或行为时间线 | 后端返回 403 或前端提示无权限；不得展示非授权数据 | □ 通过 / □ 不通过 |

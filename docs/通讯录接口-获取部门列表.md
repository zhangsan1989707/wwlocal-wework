# 获取部门列表

**请求方式：**GET
**请求地址：**https://uat.rztcd.cn/cgi-bin/department/list?access_token=ACCESS_TOKEN&id=ID&no_fetch_child=FETCH_CHILD

**参数说明 ：**

| 参数           | 必须 | 说明                                                         |
| -------------- | ---- | ------------------------------------------------------------ |
| access_token   | 是   | 调用接口凭证                                                 |
| id             | 否   | 部门id。获取指定部门及其下的子部门。 如果不填，默认获取全量组织架构 |
| no_fetch_child | 否   | 填“0”则为递归获取子部门下面的部门，填“1”则为不递归获取子部门下面的部门 |
| get_party_path | 否   | 填1，则返回传入部门的路径信息                                |

**权限说明：**

只能拉取token对应的应用的权限范围内的部门列表

**参数说明：**

| 参数                      | 说明                                                         |
| ------------------------- | ------------------------------------------------------------ |
| errcode                   | 返回码                                                       |
| errmsg                    | 对返回码的文本描述内容                                       |
| department                | 部门列表数据。                                               |
| id                        | 创建的部门id                                                 |
| name                      | 部门名称                                                     |
| parentid                  | 父部门id。根部门为1                                          |
| order                     | 在父部门中的次序值。order值大的排序靠前，order值相等的情况下，按照部门id排，id越小排序越靠前。有效的值范围是[0, 2^32) |
| party_user_limit          | 部门可添加成员上限                                           |
| type                      | 部门类型的标记。0表示普通部门，1表示实体单位；2表示虚拟单位，3表示行政区划。未赋值时，默认是普通部门 |
| party_detail              | 部门详情。type为实体单位、虚拟单位或行政区划时，该字段才有效 |
| full_name                 | 单位全称。单位内必须唯一，未赋值时，默认是部门名称           |
| short_name                | 单位简称。单位内必须唯一，未赋值时，默认是部门名称           |
| brief_introduction        | 单位简介。长度为0-256个字符                                  |
| domain_name               | 单位网站。以“www”、”http://“或”https://“开头                 |
| address                   | 单位地址。长度为0-128个字符                                  |
| telephone                 | 单位电话。由1-32位的纯数字或’-‘号组成                        |
| organization_code         | 社会信用代码，由18位大小写字母及数字组成，仅适用于部门类型为实体单位 |
| unit_admin_acct           | 单位管理员的userid，当前每个单位只能指定1个管理员            |
| superior_guidance_unit_id | 上级业务指导单位的部门id                                     |
| admin_division_id         | 所属行政区划的部门id                                         |
| admin_division_level      | 行政级别，取整数值：1: 省级 2: 地市级 3: 区县级 4: 镇乡级。type为行政区划时，该字段才有效 |
| partypath                 | 部门路径列表                                                 |
| path_info                 | 部门路径上部门信息,partyid 部门id,name 部门名称              |

**版本信息：**
v1.0以上版本有效

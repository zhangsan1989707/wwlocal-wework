# 读取成员

在通讯录同步助手中此接口可以读取单位通讯录的所有成员信息，而自建应用可以读取该应用设置的可见范围内的成员信息。默认只能读取账号字段，如需读取其他字段，可前往管理后台-应用配置页-应用安全处配置更多字段。

**请求方式：**GET
**请求地址：**https://uat.rztcd.cn/cgi-bin/user/get?access_token=ACCESS_TOKEN&userid=USERID

**参数说明：**

| 参数         | 必须 | 说明                                                         |
| ------------ | ---- | ------------------------------------------------------------ |
| access_token | 是   | 调用接口凭证                                                 |
| userid       | 是   | 成员UserID。对应管理端的账号，单位内必须唯一。不区分大小写，长度为1~64个字节 |

**权限说明：**

非通讯录同步助手的应用，需要拥有成员的权限

**参数说明：**

| 参数               | 必须 | 说明                                                         |
| ------------------ | ---- | ------------------------------------------------------------ |
| errcode            |      | 返回码                                                       |
| errmsg             |      | 对返回码的文本描述内容                                       |
| userid             |      | 成员UserID。对应管理端的账号                                 |
| name               |      | 成员名称                                                     |
| mobile             |      | 手机号码                                                     |
| hide_mobile        |      | 手机号码隐藏状态。1表示隐藏，0表示不隐藏                     |
| department         |      | 成员所属部门id列表。排序在第一个的部门id为主岗部门           |
| order              |      | 部门内的排序值，默认为0。数量必须和department一致，数值越大排序越前面。有效的值范围是[0, 2^32)。在姓名A-Z排序模式下，为置顶成员的排序值，不为0则默认置顶；在自由排序模式下，为所有成员排序值 |
| position           |      | 职位信息                                                     |
| positions          |      | 跟随部门，支持多部门不同职位信息                             |
| gender             |      | 性别。0表示未定义，1表示男性，2表示女性                      |
| email              |      | 邮箱                                                         |
| biz_mail           |      | 单位邮箱                                                     |
| biz_mail_alias     |      | 邮箱别名                                                     |
| is_leader_in_dept  |      | 表示在所在的部门内是否为上级。1表示为上级，0表示非上级。     |
| direct_leader      |      | 直属上级，数组，当前只支持1个直属上级，填 userid             |
| avatar             |      | 头像url。                                                    |
| telephone          |      | 座机                                                         |
| english_name       |      | 别名。长度不超过64个字。当别名作为登录字段时，只能由字母、数字及 .-*@ 组成， .-*@ 不能放在开头，且数据不能重复 |
| extattr            |      | 扩展属性                                                     |
| status             |      | 激活状态:激活状态: 1=已激活，2=已禁用，4=未激活              |
| enable             |      | 成员启用状态。1表示启用的成员，0表示被禁用                   |
| qr_code            |      | 成员二维码图片地址                                           |
| country_code       |      | 手机区号                                                     |
| external_profile   |      | 成员对外属性                                                 |
| external_corp_name |      | 对外简称                                                     |

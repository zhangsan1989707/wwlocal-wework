## 数据管理

使用方法：使用“数据开放服务”应用的secret获取accesstoken，相关数据参数，通过”操作日志及数据“获取。

### 1. 以群主身份解散某个群

**请求方式：**POST

**请求地址：** https://uat.rztcd.cn/cgi-bin/corp/dismiss_room?access_token=ACCESS_TOKEN

**请求参数：**

```
{  "chatid": "Gzue83uzuXzQyMOcyGq7JW3U/x",  "oper":  {    "openid": "openid",    "type": 0  }}
```

**请求参数说明：**

| 参数名称     | 是否必须 | 说明                                                         |
| ------------ | -------- | ------------------------------------------------------------ |
| access_token | 是       | 调用接口凭证                                                 |
| chatid       | 是       | 群id 获取方式参考“操作日志与数据”                            |
| oper         | 是       | 操作人(必须为本单位的用户)                                   |
| openid       | 是       | 内部成员使用userid，外部单位成员使用openid，获取方式参考“操作日志与数据” |
| type         | 是       | 0: 本单位 ,1: 互联单位                                       |

**返回结果：**

```
{  "errcode": 0,  "errmsg": "success",  "chatid": "Gzue83uzuXzQyMOcyGq7JW3U/xtLKarQjskylVHy9U4xczSBnjqxek10EN6w90N5",  "name": "room_name",  "owner":  {    "openid": "tracytest",    "type": 0  }}
```

**返回结果说明：**

| 参数名称 | 说明                                      |
| -------- | ----------------------------------------- |
| errcode  | 返回码                                    |
| errmsg   | 对返回码的文本描述内容                    |
| chatid   | 解散的群id                                |
| name     | 解散的群名                                |
| openid   | 解散群的群主信息                          |
| type     | 解散群的群主类型（0: 本单位, 1: 互联单位) |

### 2. 以群主的身份移除群成员

**请求方式：**POST

**请求地址：** https://uat.rztcd.cn/cgi-bin/corp/del_room_member?access_token=ACCESS_TOKEN

**请求参数：**

```
{  "chatid": "2gNJ2zqXPGiPx2UGkFcLk9nukSYZRN",  "oper":  {    "openid": "oper_openid",    "type": 0  },  "del_members":  [    {      "openid": "roommember1",      "type": 0    },    {      "openid": "roommember2",      "type": 0    }  ]}
```

**请求参数说明：**

| 参数名称     | 是否必须 | 说明                                                         |
| ------------ | -------- | ------------------------------------------------------------ |
| access_token | 是       | 调用接口凭证                                                 |
| chatid       | 是       | 群id，获取方式参考“操作日志与数据”                           |
| oper         | 是       | 操作人userid                                                 |
| del_members  | 是       | 需要移出的群成员                                             |
| openid       | 是       | 内部成员使用userid，外部单位成员使用openid，获取方式参考“操作日志与数据” |
| type         | 是       | 操作人的类型（0: 本单位, 1: 互联单位）                       |

**返回结果：**

```
{  "errcode": 0,  "errmsg": "success",  "chatid": "6nzQX4cAwbgPFmEtmsNgwjmAypQAjiDe",  "name": "room_name",  "owner": {    "openid": "own_openid",    "type": 0  },  "del_members":  [    {      "openid": "del_member1",      "type": 0    },    {      "openid": "del_member2",      "type": 0    }  ]}
```

**返回结果说明：**

| 参数名称    | 说明                   |
| ----------- | ---------------------- |
| errcode     | 返回码                 |
| errmsg      | 对返回码的文本描述内容 |
| name        | 移出群成员的群名       |
| owner       | 移出群成员的群主       |
| del_members | 移出成功的用户列表     |
| openid      | 用户信息               |
| type        | 0: 本单位, 1: 互联单位 |

### 3. 删除文件

**请求方式：**POST

**请求地址：** https://uat.rztcd.cn/cgi-bin/corp/del_msg_file?access_token=ACCESS_TOKEN

**请求包体：**

```
{  "sender":  {    "openid": "",    "type": 0  },  "receiver":  {    "openid": "",    "type": 0  },  "chatid": "vbTBukL+LvDRjoxWMJ6VmyEp",  "fileid": "RZz2MUygGVHkv+Fy61JDq+Ek8wIcJl5sjNkv"}
```

**参数说明：**

| 参数名称     | 是否必须 | 说明                                                         |
| ------------ | -------- | ------------------------------------------------------------ |
| access_token | 是       | 调用接口凭证                                                 |
| sender       | 是       | 消息发送者                                                   |
| receiver     | 否       | 消息接收者（单人会话填写该字段）                             |
| openid       | 是       | 内部成员使用userid，外部单位成员使用openid，获取方式参考“操作日志与数据” |
| type         | 是       | 类型（0: 本单位, 1: 互联单位）                               |
| chatid       | 否       | 群id(多人会话填写该字段)，获取方式参考“数据与安全”—“数据开放”部分 |
| fileid       | 是       | 要删除的fileid,获取方式参考“数据与安全”—“数据开放”部分       |

**返回结果：**

```
{  "errcode": 0,  "errmsg": "success"}
```

**返回结果说明：**

| 参数名称 | 说明                   |
| -------- | ---------------------- |
| errcode  | 返回码                 |
| errmsg   | 对返回码的文本描述内容 |

### 4. 删除消息

**请求方式：**POST

**请求地址：** https://uat.rztcd.cn/cgi-bin/corp/del_msg?access_token=ACCESS_TOKEN

**请求包体：**

```
{  "sender": //必填, 发送者  {    "openid": "",    "type": 0  },  "receiver": // 单聊必填, 群聊不填  {    "openid": "",    "type": 0  },  "chatid": "vbTBukL+LvDRjoxWMJ6VmyEp", //单聊不填, 群聊必填  "msgid": "RZz2MUygGVHkv+Fy61JDq+Ek8wIcJl5sjNkv"}
```

**参数说明：**

| 参数名称     | 是否必须 | 说明                                                         |
| ------------ | -------- | ------------------------------------------------------------ |
| access_token | 是       | 调用接口凭证                                                 |
| sender       | 是       | 消息发送者                                                   |
| receiver     | 否       | 消息接收者（单人会话填写该字段）                             |
| openid       | 是       | 内部成员使用userid，外部单位成员使用openid，获取方式参考“操作日志与数据” |
| type         | 是       | 类型（0: 本单位, 1: 互联单位）                               |
| chatid       | 否       | 群id(多人会话填写该字段)，获取方式参考“操作日志与数据”       |
| msgid        | 是       | 要删除消息的id,获取方式参考“操作日志与数据”                  |

**返回结果：**

```
{  "errcode": 0,  "errmsg": "success"}
```

**返回结果说明：**

| 参数名称 | 说明                   |
| -------- | ---------------------- |
| errcode  | 返回码                 |
| errmsg   | 对返回码的文本描述内容 |

### 5. 撤回消息

**请求方式：**POST

**请求地址：** https://uat.rztcd.cn/cgi-bin/corp/revoke_msg?access_token=ACCESS_TOKEN

**请求包体：**

```
{  "sender": //必填, 发送者  {    "openid": "",    "type": 0  },  "receiver": //单聊必填, 群聊不填  {    "openid": "",    "type": 0  },  "chatid": "vbTBukL+LvDRjoxWMJ6VmyEp", //单聊不填, 群聊必填  "msgid": "RZz2MUygGVHkv+Fy61JDq+Ek8wIcJl5sjNkv",  "appinfo": "",  "send_time": 0,  "revokemsg_notrace": 1,}
```

**参数说明：**

| 参数名称          | 是否必须 | 说明                                                         |
| ----------------- | -------- | ------------------------------------------------------------ |
| access_token      | 是       | 调用接口凭证                                                 |
| sender            | 是       | 消息发送者                                                   |
| receiver          | 否       | 消息接收者（单人会话填写该字段）                             |
| openid            | 是       | 内部成员使用userid，外部单位成员使用openid，获取方式参考“操作日志与数据” |
| type              | 是       | 类型（0:本单位, 1: 互联单位）                                |
| chatid            | 否       | 群id(多人会话填写该字段)，获取方式参考“操作日志与数据”       |
| msgid             | 是       | 要删除消息的id,获取方式参考“操作日志与数据”                  |
| appinfo           | 否       | 消息标识信息（发送者删除消息后，撤回原始消息时使用），获取方式参考“操作日志与数据” |
| send_time         | 否       | 消息发送时间（appinfo填了，必填），获取方式参考“操作日志与数据” |
| revokemsg_notrace | 否       | 是否无痕撤回，当且仅当revokemsg_notrace = 1为无痕撤回，其他值均为有痕撤回 |

**返回结果：**

```
{  "errcode": 0,  "errmsg": "success"}
```

**返回结果说明：**

| 参数名称 | 说明                   |
| -------- | ---------------------- |
| errcode  | 返回码                 |
| errmsg   | 对返回码的文本描述内容 |

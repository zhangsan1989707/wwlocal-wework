# 统计数据导出

## 统计数据导出

管理后台已有的统计数据导出

### 1. 应用使用分析数据

#### 1.1 按天获取应用使用分析数据

获取指定时间（按天）所有应用的使用分析统计。

**请求方式：**POST

**请求地址：** https://uat.rztcd.cn/cgi-bin/agent/app_stat?access_token=ACCESS_TOKEN

**请求包体：**

```
{    "time": 1551888000}
```

**参数说明：**

| 参数名称 | 是否必须 | 说明                                                         |
| -------- | -------- | ------------------------------------------------------------ |
| time     | 是       | 要查询日期的时间戳，会查询到时间戳当天的所有消息，最小可取值为31507200（1971-01-01 00:00:00） |

**返回结果：**

```
{    "errcode": 0,    "errmsg": "ok",    "app_list": [        {            "agentid": 1,            "name": "xxx",            "time": 1551888000,            "uv": 1,            "pv": 2,            "cnt": 3,            "msg_cnt": 100,        }    ]}
```

**返回结果说明：**

| 参数名称 | 说明                |
| -------- | ------------------- |
| errcode  | 0：正常返回 非0异常 |
| errmsg   | 错误提示信息        |
| app_list | 应用列表            |
| agentid  | 应用ID              |
| name     | 应用名称            |
| time     | 同请求中的时间戳    |
| uv       | 进入应用人数        |
| pv       | 进入应用次数        |
| cnt      | 应用总人数          |
| msg_cnt  | 当日发送消息数      |

#### 1.2获取应用消息使用分析数据

获取指定时间（按天）和应用的消息分析统计。

**请求方式：**POST

**请求地址：** https://uat.rztcd.cn/cgi-bin/agent/app_msg?access_token=ACCESS_TOKEN

**请求包体：**

```
{    "agentid": 1,    "time": 1551344787,    "keyword": "xxx",}
```

**参数说明：**

| 参数名称 | 是否必须 | 说明                                                         |
| -------- | -------- | ------------------------------------------------------------ |
| agentid  | 是       | 要查询消息所属的应用ID                                       |
| time     | 是       | 要查询日期的时间戳，会查询到时间戳当天的所有消息，最小可取值为31507200（1971-01-01 00:00:00） |
| keyword  | 否       | 要过滤的消息主题的关键字（为空或不填时查询所有消息）         |

**返回结果：**

```
{    "errcode": 0,    "errmsg": "ok",    "agentid": 1000006,    "name": "发消息",    "msg_list": [        {            "msg_type": 101, //1文字 2文件 3语音 4视频 13图片 101图文消息 102外链图文            "msg_title": "12345",            "send_time": 1570948835,            "recv_cnt": 122,            "reader_cnt": 3,            "read_cnt": 0,            "like_cnt": 3,            "like_info": [                {                    "user_name": "testtest",                    "party_name": "testcorp110204",                    "user_id": "testtest"                }            ],            "read_info": [                {                    "user_name": "testtest",                    "party_name": "testcorp110204",                    "user_id": "testtest"                },                {                    "user_name": "test5",                    "party_name": "testcorp110204",                    "user_id": "test5"                },                {                    "user_name": "test6",                    "party_name": "testcorp110204",                    "user_id": "test6"                }            ]        }    ]}
```

**返回结果说明：**

| 参数名称   | 说明                   |
| ---------- | ---------------------- |
| errcode    | 0：正常返回 非0异常    |
| errmsg     | 错误提示信息           |
| read_info  | 阅读应用消息的成员信息 |
| like_info  | 点赞应用消息的成员信息 |
| agentid    | 应用ID                 |
| name       | 应用名称               |
| msg_list   | 消息列表               |
| msg_type   | 消息类型               |
| msg_title  | 消息主题               |
| send_time  | 发送时间               |
| recv_cnt   | 送达人数               |
| reader_cnt | 通讯录内阅读人数       |
| read_cnt   | 阅读次数               |
| like_cnt   | 点赞数                 |

### 2.导出管理员操作日志

支持分页导出指定时间段内管理员操作日志

**请求方式：**POST

**请求地址：** https://uat.rztcd.cn/cgi-bin/corp/get_admin_oper_log?access_token=ACCESS_TOKEN

**请求包体：**

```
{    "start_time":0,    "end_time":0,    "start":0,    "limit":100}
```

**参数说明：**

| 参数名称   | 是否必须 | 说明                                                  |
| ---------- | -------- | ----------------------------------------------------- |
| start_time | 是       | 起始时间，时间戳                                      |
| end_time   | 是       | 结束时间，时间戳                                      |
| start      | 是       | 分页时，指定下一次获取数据的起始位置，首次获取设置为0 |
| limit      | 是       | 限制单次获取数据最大条数                              |

**返回结果：**

```
{    "errcode": 0,    "errmsg": "ok",    "oper_list": [        {            "oper_time": 1553520176,            "oper_type_id": 23,            "oper_type": "登录后台",            "oper_userid": "aaa",            "oper_name": "张三",            "oper_data": "2019-03-25 21:22"        }    ]}
```

**返回结果说明：**

| 参数名称     | 说明                |
| ------------ | ------------------- |
| errcode      | 0：正常返回 非0异常 |
| errmsg       | 错误提示信息        |
| oper_time    | 操作时间            |
| oper_type_id | 操作类型ID          |
| oper_type    | 操作类型描述        |
| oper_userid  | 操作人账号          |
| oper_name    | 操作人名称          |
| oper_data    | 操作数据            |
